# SmartAttend – Python dependencies
# Tested on CPython 3.12 (Windows 10/11 + PostgreSQL 14+).
#
# Install with:
#     python -m pip install -r requirements.txt
#     python -m pip install --no-deps face_recognition
#
# NOTE: face_recognition declares `dlib` as its dependency. On Windows we use
# the precompiled `dlib-bin` wheel instead (no C++ compiler required), so
# `face_recognition` is installed with `--no-deps` as a separate step. Pip
# will print a "missing dlib" warning – it is safe to ignore because the
# `dlib` Python module is provided by `dlib-bin`.
